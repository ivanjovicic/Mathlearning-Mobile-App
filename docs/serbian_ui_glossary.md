# Serbian UI Glossary

## Rule
Use Serbian Latin with diacritics: `č`, `ć`, `š`, `ž`, `đ`.

## Terms
- Quiz = Kviz
- Question = Pitanje
- Answer = Odgovor
- Hint = Pomoć
- Streak = Niz
- Coins = Novčići
- XP = XP
- Level = Nivo
- Mastery = Savladanost
- Progress = Napredak
- Reward = Nagrada
- Chest = Kovčeg
- Daily Run = Dnevni izazov
- Review = Ponavljanje
- Settings = Podešavanja
- Feedback = Povratna informacija
- Combo = Niz pogodaka (ili Niz, po kontekstu)
